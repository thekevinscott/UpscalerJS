This page runs a speed test on UpscalerJS.

Each model has some inherent amount of latency to run. We want to ensure that the surrounding UpscalerJS does not add significant latency to the baseline model latency.

This page runs two tests: it runs a basic model with minimal latency that simply scales up an image without upscaling, and then it runs a version of the GANS model. For each test, it first runs the model raw (e.g., by calling model.predict directly), and then again using UpscalerJS.
